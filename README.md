# CYB 213: Basic Network Threat Classifier

Python-based ML for cybersecurity: Binary classification of network traffic using KDD Cup 99 dataset.

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Download dataset to `data/kddcup.data_10_percent` from [here](https://kdd.ics.uci.edu/databases/kddcup99/kddcup.data_10_percent.gz)
3. Run: `python src/network_threat_classifier.py`

## Results
- Accuracy: ~0.99
- See console output for matrix/report.
